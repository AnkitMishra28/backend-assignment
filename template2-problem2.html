Template 2 — Compound Interest (Quarterly)
Given: P = ₹2000
r = 8% p.a., n = 2 years, comp quarterly
Find Amount
Quarterly rate = 8/4 = 2% per quarter, number of quarters = 2*4 = 8
A = P(1 + r_q/100)^8 = 2000(1 + 2/100)^8
= 2000 * (1.02)^8
≈ 2000 * 1.171659 = 2343.318
Amount ≈ ₹2343.32
Answer: A ≈ ₹2343.32
